## MediVoice AI – AI Medical Voice Assistant

MediVoice AI is an intelligent healthcare web application designed to provide real-time medical assistance using voice-based interaction. The system allows users to describe their symptoms through voice or text input and leverages AI to analyze the information, suggest relevant medical specialists, and generate structured medical reports.

The platform aims to improve accessibility to healthcare guidance by reducing dependency on manual consultations for initial diagnosis and providing instant, AI-driven insights.

Features
Voice-based symptom input
AI-powered symptom analysis
Smart specialist recommendation
Automated medical report generation
Consultation history tracking
Secure authentication system
Free & Premium subscription model

Tech Stack
Frontend: React.js
Speech-to-Text: AssemblyAI
AI Voice Interaction: Vapi.ai
Database: Neon DB (PostgreSQL)
Authentication: Clerk

How It Works
User logs into the application
Starts a consultation session
Provides symptoms via voice or text
AI processes input and suggests a specialist
AI interacts with the user for further details
System generates a structured medical report


Objective

The goal of MediVoice AI is to provide a fast, accessible, and intelligent healthcare support system that assists users in understanding their symptoms and receiving initial medical guidance anytime, anywhere.
